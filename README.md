# KnowOneActual Homebrew Tap

This is a custom Homebrew tap for tools, scripts, and applications. 

## How to Use This Tap

To install packages from this repository, you first need to add the tap to your local Homebrew installation. Run this command in your terminal:

```bash
brew tap knowoneactual/tap

```

## Available Formulas

Currently, this tap includes the following tools:

* **[Isobar](https://github.com/KnowOneActual/isobar-cli):** A terminal weather tool focused on what it *feels like* outside.

## Installing a Tool

Once you have added the tap, you can install any of the available formulas using the standard `brew install` command. For example, to install Isobar, run:

```bash
brew install isobar

```

## Documentation

For full documentation, configuration options, and source code for any specific tool, please visit its respective linked repository.
